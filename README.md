## BUCT_THEOL_PDF

北京化工大学在线教育平台pdf文档下载，无视下载权限。下载下来的是教师实际上传的文档，不是图片。
## 支持范围

目前仅支持https://course.buct.edu.cn/meol/common/script/preview/* 范围下的PDF文档，后续会支持其他类型的。

## 使用方式

添加油猴脚本，点击右下角的下载按钮即可。
[](https://greasyfork.org/zh-CN/scripts/469075-buct-theol-pdf)
